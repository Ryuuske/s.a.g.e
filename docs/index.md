# docs/index.md — documentation registry

One row per doc surface. "One fact, one home": the Scope column names what
each doc OWNS — no other doc may assert those facts. Generated docs are
regenerated by `scripts/gen_docs.py` / `scripts/gen_adr_index.py`; the
regen-diff gate fails CI on drift. Countable facts (agent/skill/command
counts) live ONLY in generated docs.

## Key files for common tasks

- **Adding an MCP tool**: `src/sage_mcp/mcp_server.py` — handler + TOOLS dict entry.
- **Changing search**: `src/sage_mcp/searcher.py` · **Mining**: `src/sage_mcp/miner.py` / `convo_miner.py`.
- **Storage backend**: subclass `src/sage_mcp/backends/base.py`, register in `backends/__init__.py`.
- **Input validation**: `src/sage_mcp/config.py` — `sanitize_name()` / `sanitize_content()`.
- **Verdict schema**: `docs/specs/verdict-schema.md` + `src/sage_mcp/verdict_parser.py`.
- **Telemetry**: `docs/specs/telemetry.md` + `src/sage_mcp/telemetry.py`.
- **Ship surface**: `src/sage_mcp/export.py` allowlist + `tests/test_export.py`.
